Thinclient Software – Modifications

 

          As you are aware, the entire operations under the Credit Guarantee Scheme are in electronic mode (Online).  Besides enabling the Operating Offices of our Member Lending Institutions (MLIs) to lodge applications online by allotting them a Member-id, User-id and Passwords, CGTSI had developed a customized software "Thinclient" to facilitate submission of applications in batch mode. Using the ThinClient software, the Operating Offices can first fill in the required information, as per the format provided, for each of the loan proposal and then upload all the applications, irrespective of number of such applications, at the same time to CGTSI website.  This software was made available to MLIs, their Operating Offices and is also available for downloading at CGTSI website.

 

          A few MLIs have brought to our notice certain limitations in the existing Thinclient software, viz. inability to run on certain windows platforms like Windows 2000, Windows XP, the obsolete IP address which required to be modified by the MLIs, not able to accept applications where sanction amount is above Rs. 25 lakh, etc.

         

         In order to remove those shortcomings and to make the software more user-friendly, Thinclient software has since been modified and the new version has the following salient features:

 

1.      Runs on Windows 98, Windows NT, Windows 2000 and Windows XP platforms

 

2.      IP address is changed to 210.210.17.65

 

3.      MLIs can lodge applications where sanction amount is above Rs. 25 lakh

 

4.      A new Industry Type viz. "Services (Industry Related)" (SSSBEs) has been added to the list of "Industry Type" field, etc. 

 

5.      The installation procedure is simplified and creates a short-cut on the desktop for easy access.

 

          Though the MLIs can continue to use the existing Thinclient software, use of the new version will become necessary when any application pertaining to SSSBEs has to be submitted to CGTSI, or the MLI operates on platforms like Windows 2000 or Windows XP.  We hereby request the MLIs to download the new version of Thinclient software by logging in to the website of CGTSI using the Member ID, User ID and Password.  In case of any difficulty please inform us to enable us to provide floppies to the interested Operating Offices.

 

          We also request you to suitably inform your Operating Offices about the above modifications.

 

          Thanking you for your cooperation,
