Credit Guarantee Scheme (CGS) – New organization - wide

IT System – Problems in Login

 

            As the operating offices of the Member Lending Institutions (MLIs) may be aware, CGTSI had effected a switch-over from the present online system to a new, improved and feature rich online system in the first week of April 2005 [Ref: CGTSI Circular no. 22 / 2004-05 dated March 15, 2005].  Along with the salient features of the New System, the procedure for logging into the new system has been given in the aforesaid Circular. It is observed, however, that some of the operating offices of MLIs have been facing problems in logging-in and hence, they have been contacting CGTSI for the purpose. It may be mentioned that though MLI-users were expected to login into the CGTSI web-site by using their Member ID [existing 12-digit code], User Id [existing user Id] and the Password [the word ‘password’ in lower case], some of the MLI-users have / had used their existing passwords thereby getting their ‘User IDs’ locked. In order to properly access the New System of CGTSI the MLI-users are requested to observe the following procedure:

 

ü      Go to the CGTSI web-site [www.cgtsi.org.in]

 

ü      Click on ‘MEMBERS LOGIN’ (A help screen (Trouble Logging In?) would be available for your guidance.  You can click Member’s Login in that again). 

 

ü      In the next screen, in the blank space against Member ID, User ID and the Password, key-in your existing Member ID, existing User ID and type the word ‘password’ (in lower case characters) in the box meant for password.  This is the default password (instead of your existing password).  After this click on the Submit Button.  [Kindly note that if the existing password is typed here instead of the word ‘password’, you will not be able to log in.]

 

ü      Another Screen will get opened prompting you to change the password.

 

  

Please proceed as under and then click Save button.

 

 
Change Password
 *Old Password -- Type the word 'password'
 *New Password -- Give a new password of your choice
 *Confirm New Password -- Re-type the new password entered above
 *Enter New Hint Question -- Type any hint-question of your choice
 *Enter Hint Answer -- Type the answer you want to set for the 
                          hint question above
  
 

 

ü      If the above procedure is correctly followed the MLI operating office’s Home page on CGTSI will get opened and applications can be lodged as well as all other facilities / information accessed. 

 

            The MLI-operating offices that are already using the New System as per the instructions given in the CGTSI Circular No. 22 / 2004-05 and have already logged-in may kindly ignore this Circular.  They would be able to access the system by clicking on the "Click here" hyperlink or 'Members Login' button available in the help screen (Trouble Loggin In?)

 

            Please feel free to get in touch with us in case any further clarifications are required.
