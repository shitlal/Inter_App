Trouble Logging In?

Kindly ignore this procedure if you have already logged into new system successfully.  to proceed or click on the 'Members Login' button available at the bottom of the page.

Please follow the procedure as detailed below if you are not able to login into the system:

    Important: Click on the Members Login button (available at the bottom of the page) after reading the following.

    A screen appears with 3 boxes for entering Member ID, User ID and Password.

    Key-in your existing 12-digit Member ID, existing User ID and type the word 'password' (in lower case characters) in the box meant for password.  This is the default password (instead of your existing password).

    Another screen appers with 5 boxes prompting you to change the password. 

    In the first box (Old Password), key-in the word 'password' (in lower case characters). 

    In the second box (New Password), key-in any password as you desire and in the third box, re-type the same password. 

    Fourth box is where you enter your own 'Hint Question' and in the fifth box, enter 'Hint Answer'.  

    Click on 'Save' button.  Then the Member Home page appears and applications can be lodged as well as all other facilities and information can be accessed.
